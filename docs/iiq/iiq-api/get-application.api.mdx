## Returns an Application resource based on id or name.



The Application resource with matching id or name is returned. Attributes to include in the response can be specified with the &#39;attributes&#39; query parameter. 
Attributes to exclude from the response can be specified with the &#39;excludedAttributes&#39; query parameter. 
The schema related to Application is: 
- **urn:ietf:params:scim:schemas:sailpoint:1.0:Application.**
