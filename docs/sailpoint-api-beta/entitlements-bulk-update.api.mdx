## Bulk update an entitlement list



This API applies an update to every entitlement of the list.

The number of entitlements to update is limited to 50 items maximum.

The JsonPatch update follows the [JSON Patch](https://tools.ietf.org/html/rfc6902) standard.&lt;br/&gt; allowed operations :&lt;br/&gt; **{ &quot;op&quot;: &quot;replace&quot;, &quot;path&quot;: &quot;/privileged&quot;, &quot;value&quot;: boolean }** &lt;br/&gt; **{ &quot;op&quot;: &quot;replace&quot;, &quot;path&quot;: &quot;/requestable&quot;,&quot;value&quot;: boolean }** &lt;br/&gt;

A token with ORG_ADMIN or API authority is required to call this API.
