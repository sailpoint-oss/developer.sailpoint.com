## Get Summary of Non-Employee Approval Requests



This request will retrieve a summary of non-employee approval requests.&lt;br&gt;&lt;br&gt; There are two contextual uses for the `requested-for` path parameter:&lt;br&gt;
  1. The user has the role context of `idn:nesr:read`, in which case he or she may request a summary of all non-employee approval requests assigned to a particular approver by passing in that approver&#39;s id.&lt;br&gt;
  2. The current user is an approver, in which case &quot;me&quot; should be provided as the `requested-for` value. This will provide the approver with a summary of the approval items assigned to him or her.
