Plugins for IdentityIQ extend the core product to solve additional use cases.

## Install a plugin

1. Download the plugin `.zip` file you want from CoLab **IIQ Plugins**.
2. Log in to IdentityIQ as an administrator (or a user with **Plugin Administrator** capability).
3. Select the gear icon and open **Plugins**.
4. Click **New**.
5. Upload the `.zip` file by drag-and-drop or file picker.
6. Verify installation completed successfully.
7. Configure plugin-specific settings from the plugin's **Configure** action.

## Uninstall a plugin

1. Log in as an administrator (or user with **Plugin Administrator** capability).
2. Open **Plugins** from the gear icon.
3. Locate the plugin and click the **X** action.
4. Confirm removal.

For additional installation and configuration help, see the IdentityIQ product documentation.
