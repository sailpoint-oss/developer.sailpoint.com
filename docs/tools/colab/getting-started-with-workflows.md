Use this category to view and download Identity Security Cloud (ISC) workflows built by the community.

Follow these steps to download a workflow from CoLab and start using it:

1. Download the workflow you want from the **Workflows** category.
2. Go to the Workflows dashboard in your tenant and click **+ New Workflow**.
3. Choose **Start with a JSON file**.
4. Drag and drop your JSON file into the upload window, or browse for the file.
5. Update the workflow name if needed.
6. Enter an optional description.
7. Click **Upload New Script**.

Once you upload the JSON file, you can start using the workflow in your tenant.

:::note
After saving your workflow, test it before enabling it in production.
:::
