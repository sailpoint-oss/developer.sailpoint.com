Requestability

Properties​

NameTypeDescriptionNotes
CommentsRequiredPointer to NullableBoolIndicates whether the requester of the containing object must provide comments justifying the request.[optional] [default to false]
DenialCommentsRequiredPointer to NullableBoolIndicates whether an approver must provide comments when denying the request.[optional] [default to false]
ReauthorizationRequiredPointer to NullableBoolIndicates whether reauthorization is required for the request.[optional] [default to false]
RequireEndDatePointer to NullableBoolIndicates whether the requester of the containing object must provide access end date.[optional] [default to false]
MaxPermittedAccessDurationPointer to NullableAccessduration[optional]
ApprovalSchemesPointer to []AccessprofileapprovalschemeList describing the steps involved in approving the request.[optional]
